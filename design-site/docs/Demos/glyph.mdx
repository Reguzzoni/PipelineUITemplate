# Algo Glyph

Algo Glyph is an application that allows users to create a real signature to solidify their digital identity. AlgoGlyph signatures are stored on-chain, forever.

:::info

This demo is an embedded iFrame. To test AlgoSigner functionality, please visit [this link](https://algo-glyph.vercel.app).

:::
