# Blockie

Blockies are identicons for addresses and other lengthy nonsemantic alphanumeric data. They are not meant to replace user profiles. Instead, they allow users to check if an address they want to interact with is the correct one.

import {Blockie} from 'pipeline-ui'

## Usage Example

```jsx live
<Blockie
  opts={{
    seed: "LMKFQIPL3VQCZGGFK4WZ7FPCQWLNBTJQ3UWSTA7D7QZSPJTZQKTDVT7WG4",
    color: "#dfe",
    bgcolor: "#a71",
    size: 40,
    scale: 3,
    spotcolor: "#000"
  }}
/>
```
