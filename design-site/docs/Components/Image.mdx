# Image

Use the Image component to display an image in your product. This renders the `<img>` HTML element and extends the Box component.

import {Image} from 'pipeline-ui'

## Usage Example

```jsx live
<Image
  alt="Algorand"
  borderRadius={8}
  height="auto"
  src="https://www.algorand.com/assets/media-kit/logos/full/png/algorand_full_logo_black.png"
/>
```
