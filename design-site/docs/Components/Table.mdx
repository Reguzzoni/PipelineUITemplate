# Table

Table renders the `<table>` HTML element and inherits props from the Box component.

import {Table} from 'pipeline-ui'

## Usage Example

```jsx live
<Table>
  <thead>
    <tr>
      <th>Transaction ID</th>
      <th>Value</th>
      <th>Recipient</th>
      <th>Time</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>K4D...FGE</td>
      <td>0.10 Algo</td>
      <td>ER6...JSD</td>
      <td>July 31 2021 12:47:17 AM +UTC</td>
    </tr>
    <tr>
      <td>S3G...230</td>
      <td>0.11 Algo</td>
      <td>4GJ...1E1</td>
      <td>July 31 2021 12:52:17 AM +UTC</td>
    </tr>
    <tr>
      <td>8H6...C40</td>
      <td>0.12 Algo</td>
      <td>6H6...93J</td>
      <td>July 31 2021 12:55:17 AM +UTC</td>
    </tr>
  </tbody>
</Table>
```
