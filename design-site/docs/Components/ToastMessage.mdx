# ToastMessage

Use toast messages to provide updates about a process. The ToastMessage component extends Box.

import {ToastMessage} from 'pipeline-ui'

Usage Example:

```jsx live
<ToastMessage.Failure
  my={3}
  message={"Payment failed"}
  secondaryMessage={"You don't have enough Exodus tokens"}
  actionText={"Buy tokens"}
  actionHref={"#!"}
/>
```
